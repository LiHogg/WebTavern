# Демо-данные для локального запуска

После применения миграций можно загрузить стартовый демонстрационный набор:

```bash
python manage.py seed_demo_data
```

Если нужно полностью пересобрать именно демо-набор:

```bash
python manage.py seed_demo_data --reset-demo
```

## Что создаётся

- 6 тестовых аккаунтов: владелец, менеджер, 2 клиента, модератор, администратор платформы;
- 1 юридическое лицо владельца;
- 2 заведения с разными статусами;
- 3 зала;
- набор столов и координат на схемах;
- правила бронирования и брендирование заведений;
- назначения менеджера;
- несколько броней с разными статусами;
- платежи для части броней;
- уведомления;
- запись в лист ожидания.
